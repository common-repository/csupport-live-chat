== Description ==
Note: This plugin requires a cSupport Live Chat account. You can signup for a free 30-day trial at [csupporthq.com/pricing](https://csupporthq.com/pricing)

Extend the human contact. Talk directly with your website visitors!

This plugin implements an inline *Live Chat* button to the bottom right of your WordPress site.

Live Chat features include:

+ No client software to download
+ Service live chats from any browser, including smart phones
+ Service multiple visitors simultaneously
+ Canned responses (pre-configured text added to a chat with a single click or short-code)
+ Includes a contact form when you are not available/online to serve chats
+ Google Talk Integration to chat with your website visitors directly from your google talk client
+ Reliable and secure
+ Complete chat history available for visitors and for the website administrator
+ The live chat service is provided by csupporthq.com. See [www.csupporthq.com](https://csupporthq.com/) for a full list of features.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `cSupportLivechat.php` to the `/wp-content/plugins/` directory
1. Activate the Live Chat by cSupport plugin through the 'Plugins' menu in WordPress
